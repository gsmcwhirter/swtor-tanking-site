# swtor-tanking site

This is a website that employs the optimizer from swtor-apps/swtor-tanking

## Installation

0. Install node and npm
1. Clone this repository
2. Run `npm install -g grunt`
3. Run `npm install`
4. Run `grunt forever:start`
5. Visit http://127.0.0.1:8004
